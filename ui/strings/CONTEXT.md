# Where each string appears

Generated by `tools/i18n_context.py` -- do not edit.

Use this when translating: the same English word is often a different part of
speech in different places. In this UI **Home** is a verb (drive a reel to its zero
position), **Flap** is a single character leaf on a reel, **Display** is both a tab
and the wall itself, and **Save** is a button while **Serial Number** and **Last
Seen** are table headings.

This gateway *draws* the split-flap wall on an LED matrix instead of driving real
modules, so keep two things distinct: **Panel** is the physical HUB75 LED hardware
(its width, height, brightness, color depth), while **Module**, **Flap** and
**Wall** are the simulated split-flap units painted onto it.

The base English is **US spelling** (`color`, not `colour`). An en-US browser resolves
to this base; `en-GB` and `en-AU` are thin spelling diffs layered over it.
